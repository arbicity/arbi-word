# ARBI for Microsoft Word

AI-powered assistant for Microsoft Word that helps you write, summarize, and improve your documents.

## Features

- **Summarise** - Condense selected text into key points
- **Get Cites** - Find citations for selected text  
- **AutoComplete** - Continue writing from cursor position

## Installation

### Quick Install

1. Download the manifest: [arbi-word-addin.xml](https://raw.githubusercontent.com/arbitrationcity/arbi-word/main/arbi-word-addin.xml)
2. Open Word (Desktop or Online)
3. Go to **Insert → Add-ins → My Add-ins**
4. Click **Upload My Add-in**
5. Select the downloaded `arbi-word-addin.xml` file
6. ARBI will appear in your Home ribbon

### Using the Add-in

1. **First time setup**: Click the ARBI "AI Assistant" button in the ribbon to log in
2. **Right-click any selected text** to see ARBI options:
   - Summarise
   - Get Cites
   - AutoComplete
